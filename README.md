## Project Overview

This repository contains a Jupyter Notebook that performs a data analysis on student performance metrics. The goal is to:

- Explore which features (e.g., study time, past failures, demographic characteristics) relate most strongly to student success.
- Build predictive models to forecast student performance (grades, pass/fail, etc.).
- Provide actionable insights that could help educators or administrators.

---

## Problem Statement

Understanding what factors contribute to student performance can help in identifying the students who need additional support and designing interventions. This project tries to answer:

- Which factors (study habits, previous performance, personal attributes) are most predictive of student success?  
- Can we reliably predict whether a student will perform well or poorly?  
- What are key levers (features) that can be used to improve performance?
